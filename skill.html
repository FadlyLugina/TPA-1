Program

HTML



HTML stands for Hypertext Markup Language. It is a markup language used to create and structure content on the web. HTML is the standard language used to create web pages and is the foundation of the World Wide Web.

CSS



CSS stands for Cascading Style Sheets. It is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS is used to add style and formatting to web pages, including layout, colors, fonts, and more.

Javascript



JavaScript is a high-level programming language that is commonly used for creating interactive and dynamic web pages. It is an interpreted language, which means that it is executed directly by the web browser without the need for a separate compilation step.

Adobe

Adobe Photoshop



Adobe Photoshop is a graphics editing software developed by Adobe Inc. It is widely used by designers, photographers, and artists to edit and manipulate digital images, create digital art, and design graphics for print and web.

Adobe Illustrator



Adobe Illustrator is a vector graphics editing software developed by Adobe Inc. It is used by designers, artists, and illustrators to create graphics, logos, illustrations, and other visual content for print and web.

Adobe After Effect



Adobe After Effects is a digital visual effects, motion graphics, and compositing software developed by Adobe Inc. It is used by video editors, motion graphics designers, and visual effects artists to create visual effects and motion graphics for film, television, video, and the web.

Ms. Office

Microsoft Word



Microsoft Word is used to create and edit text-based documents, such as reports, letters, resumes, and more. It provides a range of formatting tools that allow users to customize the appearance of their documents, including font styles, sizes, and colors, paragraph formatting, page layout, and more.

Microsoft PowerPoint



Microsoft Excel is used to organize, analyze, and manipulate data in a spreadsheet format. It provides a range of tools and functions for working with numerical data, such as formulas, charts, and graphs. Users can create spreadsheets that perform calculations, organize data, and generate reports based on the data entered.

Microsoft Excel



Microsoft PowerPoint is used to create and deliver presentations in a variety of settings, including business meetings, educational settings, and conferences. It provides a range of tools for creating slides that include text, images, charts, graphs, and other multimedia elements.
